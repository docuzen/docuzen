# docuzen Desktop App

This workspace contains the Tauri v2 desktop shell, Vite frontend, and dev
launcher for docuzen.

Run it from this directory:

```bash
npm run tauri dev
```

The dev command starts the `@ai-native-doc/docd` sidecar and Vite before opening
the native Tauri window. For complete install, model, web search, usage, build,
and troubleshooting instructions, see the repository docs:

- [`../../README.md`](../../README.md)
- [`../../docs/install.md`](../../docs/install.md)
